# GoAdmin + Gin

一个基于 GoAdmin + Gin 的后台管理系统示例项目

------------------------------------------------------------------------

## 🚀 技术栈

-   Go
-   Gin
-   GoAdmin
-   MySQL 8
-   AdminLTE Theme

------------------------------------------------------------------------

## 📦 项目结构

    project/
    ├─ main.go
    ├─ go.mod
    ├─ sql/
    │   └─ goadmin.sql
    └─ README.md

------------------------------------------------------------------------

## ⚙️ 环境要求

-   Go \>= 1.23
-   MySQL \>= 8.0
-   Redis（可选，用于 session）

------------------------------------------------------------------------

## 🛠️ 安装步骤

### 1️⃣ 安装依赖

``` bash
go mod tidy
```

------------------------------------------------------------------------

### 2️⃣ 启动 MySQL

创建数据库：

``` sql
CREATE DATABASE goadmin DEFAULT CHARSET utf8mb4;
```

------------------------------------------------------------------------

### 3️⃣ 导入 GoAdmin 数据表

``` bash
mysql -uroot -proot goadmin < sql/goadmin.sql
```

------------------------------------------------------------------------

### 4️⃣ 修改数据库配置

在 main.go：

``` go
Databases: map[string]adm.Database{
    "default": {
        Host:   "127.0.0.1",
        Port:   "3306",
        User:   "root",
        Pwd:    "root",
        Name:   "goadmin",
        Driver: "mysql",
    },
},
```

------------------------------------------------------------------------

### 5️⃣ 启动项目

``` bash
go run main.go
```

------------------------------------------------------------------------

## 🌐 访问地址

http://localhost:9033/admin/login

------------------------------------------------------------------------

## 🔐 默认登录方式

⚠️ 本项目没有默认账号，需要手动初始化：

``` sql
INSERT INTO goadmin_users (id, username, password, name)
VALUES (1, 'admin', 'admin', 'Administrator');
```
